# Статистика

## Общая статистика

```java
Parameters params = new Parameters();
params.build("tag", "welcome");

OutboundStats stats = client.getOutboundStats(params);
System.out.println("Отправлено: " + stats.getSent());
System.out.println("Отказов: " + stats.getBounced());
System.out.println("Открытий: " + stats.getOpens());
System.out.println("Кликов: " + stats.getTotalClicks());
```

## Статистика отправок по дням

```java
OutboundSendStats sendStats = client.getSendStats(new Parameters());
for (SentStat day : sendStats.getDays()) {
    System.out.println(day.getDate() + ": " + day.getSent());
}
```

## Статистика отказов по дням

```java
OutboundBounceStats bounceStats = client.getBounceStats(new Parameters());
```

## Статистика спама

```java
OutboundSpamStats spamStats = client.getSpamStats(new Parameters());
```

## Статистика открытий

```java
// По дням
OutboundOpenStats openStats = client.getOpenStats(new Parameters());

// По платформам
OutboundOpenPlatformStats platformStats = client.getOpenPlatformStats(new Parameters());
```

## Статистика кликов

```java
// По дням
OutboundClickStats clickStats = client.getClickStats(new Parameters());

// По расположению ссылок
OutboundClickLocationStats locationStats = client.getClickLocationStats(new Parameters());

// По платформам
OutboundClickPlatformStats platformStats = client.getClickPlatformStats(new Parameters());
```

## Параметры фильтрации

Все методы статистики поддерживают следующие параметры:

| Параметр | Тип | Описание |
|---|---|---|
| `tag` | String | Фильтр по тегу |
| `fromdate` | Date | Начальная дата |
| `todate` | Date | Конечная дата |
| `messagestream` | String | Фильтр по каналу |
