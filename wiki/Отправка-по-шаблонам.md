# Отправка по шаблонам

## Управление шаблонами

### Создание шаблона

```java
import ru.haskimail.client.data.model.templates.Template;

Template template = new Template();
template.setName("Приветственное письмо");
template.setSubject("Добро пожаловать, {{name}}!");
template.setHtmlBody("<h1>Привет, {{name}}!</h1><p>Спасибо за регистрацию.</p>");
template.setTextBody("Привет, {{name}}! Спасибо за регистрацию.");

Template created = client.createTemplate(template);
System.out.println("ID шаблона: " + created.getTemplateId());
```

### Получение шаблона

```java
// По ID
Template template = client.getTemplate(12345);

// По алиасу
Template template = client.getTemplate("welcome");
```

### Обновление шаблона

```java
template.setSubject("Обновлённая тема");
client.setTemplate(template.getTemplateId(), template);
```

### Удаление шаблона

```java
client.deleteTemplate(12345);
// или по алиасу
client.deleteTemplate("welcome");
```

### Список шаблонов

```java
Parameters params = new Parameters();
params.build("count", 50).build("offset", 0);

Templates templates = client.getTemplates(params);
```

## Валидация шаблона

```java
import ru.haskimail.client.data.model.templates.TemplateToValidate;
import ru.haskimail.client.data.model.templates.TemplateValidation;

TemplateToValidate toValidate = new TemplateToValidate();
toValidate.setSubject("Тема: {{subject}}");
toValidate.setHtmlBody("<p>{{body}}</p>");

TemplateValidation result = client.validateTemplate(toValidate);
System.out.println("Валидно: " + result.getAllContentIsValid());
```

## Отправка по шаблону

```java
import ru.haskimail.client.data.model.templates.TemplatedMessage;
import java.util.HashMap;

TemplatedMessage message = new TemplatedMessage();
message.setTemplateId(12345);
message.setFrom("от@домен.ru");
message.setTo("кому@домен.ru");

HashMap<String, Object> model = new HashMap<>();
model.put("name", "Иван");
model.put("order_id", "ORD-123");
message.setTemplateModel(model);

// Дополнительные настройки
message.setTrackOpens(true);
message.setMessageStream("outbound");

MessageResponse response = client.deliverMessageWithTemplate(message);
```

## Пакетная отправка по шаблонам

```java
List<TemplatedMessage> messages = new ArrayList<>();

TemplatedMessage msg1 = new TemplatedMessage();
msg1.setTemplateAlias("welcome");
msg1.setFrom("от@домен.ru");
msg1.setTo("user1@test.ru");
msg1.setTemplateModel(Map.of("name", "Иван"));
messages.add(msg1);

TemplatedMessage msg2 = new TemplatedMessage();
msg2.setTemplateAlias("welcome");
msg2.setFrom("от@домен.ru");
msg2.setTo("user2@test.ru");
msg2.setTemplateModel(Map.of("name", "Мария"));
messages.add(msg2);

List<MessageResponse> responses = client.deliverMessageWithTemplate(messages);
```
