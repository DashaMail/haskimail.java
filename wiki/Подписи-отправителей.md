# Подписи отправителей

Управление подписями отправителей через `AccountApiClient`.

## Получение списка подписей

```java
import ru.haskimail.client.data.model.senders.*;

Signatures signatures = client.getSenderSignatures(new Parameters());
```

## Детали подписи

```java
SignatureDetails details = client.getSenderSignatureDetails(12345);
```

## Создание подписи

```java
SignatureToCreate signature = new SignatureToCreate();
signature.setFromEmail("noreply@example.ru");
signature.setName("Мой сервис");
signature.setReplyToEmail("support@example.ru");

SignatureDetails created = client.createSenderSignature(signature);
```

## Обновление подписи

```java
SignatureToCreate update = new SignatureToCreate();
update.setName("Обновлённое имя");
client.setSenderSignature(signatureId, update);
```

## Верификация SPF

```java
client.verifySenderSignatureSPF(signatureId);
```

## Повторная отправка подтверждения

```java
client.resendSenderSignatureConfirmation(signatureId);
```

## Удаление подписи

```java
client.deleteSenderSignature(12345);
```
