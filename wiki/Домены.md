# Домены

Управление доменами осуществляется через `AccountApiClient`.

## Получение списка доменов

```java
import ru.haskimail.client.data.model.domains.*;

Domains domains = client.getDomains();
```

## Детали домена

```java
DomainDetails details = client.getDomainDetails(12345);
System.out.println("DKIM верифицирован: " + details.getDkimVerified());
System.out.println("SPF верифицирован: " + details.getSpfVerified());
```

## Создание домена

```java
Domain domain = new Domain();
domain.setName("example.ru");

DomainDetails created = client.createDomain(domain);
```

## Верификация

```java
// Проверка SPF
client.verifyDomainSPF(domainId);

// Проверка DKIM
client.verifyDomainDKIM(domainId);

// Проверка Return-Path
client.verifyDomainReturnPath(domainId);

// Ротация DKIM
client.rotateDomainDKIM(domainId);
```

## Удаление домена

```java
client.deleteDomain(12345);
```
