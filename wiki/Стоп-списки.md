# Стоп-списки (Suppressions)

Стоп-списки позволяют управлять адресами электронной почты, на которые запрещена отправка сообщений. Каждый стоп-список привязан к определённому каналу.

## Получение записей стоп-списка

```java
import ru.haskimail.client.data.model.suppressions.*;

Parameters params = new Parameters();
params.build("SuppressionReason", "ManualSuppression");

Suppressions suppressions = client.getSuppressions("outbound", params);

for (Suppression s : suppressions.getSuppressions()) {
    System.out.println(s.getEmailAddress() + " - " + s.getSuppressionReason());
}
```

### Причины блокировки

| Причина | Описание |
|---|---|
| `ManualSuppression` | Добавлено вручную |
| `HardBounce` | Жёсткий отказ доставки |
| `SpamComplaint` | Жалоба на спам |

## Добавление в стоп-список

```java
SuppressionEntries entries = new SuppressionEntries("spam@example.com", "bounce@example.com");
SuppressionStatuses result = client.createSuppressions("outbound", entries);

for (SuppressionStatus status : result.getSuppressions()) {
    System.out.println(status.getEmailAddress() + ": " + status.getStatus());
}
```

## Удаление из стоп-списка

```java
SuppressionEntries entries = new SuppressionEntries("restored@example.com");
SuppressionStatuses result = client.deleteSuppressions("outbound", entries);
```
