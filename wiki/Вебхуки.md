# Вебхуки

Вебхуки позволяют получать уведомления о событиях (доставка, открытие, клик, отказ и т.д.) в реальном времени.

## Получение списка вебхуков

```java
Webhooks webhooks = client.getWebhooks(new Parameters());
```

## Получение вебхука по ID

```java
Webhook webhook = client.getWebhook(12345);
```

## Создание вебхука

```java
import ru.haskimail.client.data.model.webhooks.*;

Webhook webhook = new Webhook("https://ваш-домен.ru/webhook");

// Настройка триггеров
WebhookTriggers triggers = new WebhookTriggers();
webhook.setTriggers(triggers);

// HTTP-аутентификация (опционально)
HttpAuth auth = new HttpAuth();
auth.setUsername("user");
auth.setPassword("pass");
webhook.setHttpAuth(auth);

// Привязка к каналу
webhook.setMessageStream("outbound");

Webhook created = client.createWebhook(webhook);
```

## Обновление вебхука

```java
webhook.setUrl("https://new-url.ru/webhook");
client.setWebhook(webhook.getId(), webhook);
```

## Удаление вебхука

```java
client.deleteWebhook(12345);
```
