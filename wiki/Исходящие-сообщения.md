# Исходящие сообщения

## Получение списка сообщений

```java
Parameters params = new Parameters();
params.build("count", 50).build("offset", 0);

OutboundMessages messages = client.getMessages(params);
```

## Детали сообщения

```java
OutboundMessageDetails details = client.getMessageDetails("message-id");
System.out.println("Тема: " + details.getSubject());
System.out.println("Статус: " + details.getStatus());
```

## Дамп сообщения

```java
OutboundMessageDump dump = client.getMessageDump("message-id");
System.out.println(dump.getBody());
```

## Открытия сообщений

```java
// Все открытия
OutboundMessageOpens opens = client.getMessageOpens(new Parameters());

// Открытия конкретного сообщения
OutboundMessageOpens opens = client.getMessageOpens("message-id", new Parameters());
```

## Клики по ссылкам

```java
// Все клики
OutboundMessageClicks clicks = client.getMessageClicks(new Parameters());

// Клики конкретного сообщения
OutboundMessageClicks clicks = client.getMessageClicks("message-id", new Parameters());
```
