# Отказы доставки (Bounces)

## Получение списка отказов

```java
import ru.haskimail.client.Parameters;
import ru.haskimail.client.data.model.bounces.*;

Parameters params = new Parameters();
params.build("count", 25)
      .build("offset", 0)
      .build("type", "HardBounce");

Bounces bounces = client.getBounces(params);
System.out.println("Всего отказов: " + bounces.getTotalCount());

for (Bounce bounce : bounces.getBounces()) {
    System.out.println(bounce.getEmail() + " - " + bounce.getType());
}
```

### Параметры фильтрации

| Параметр | Тип | Описание |
|---|---|---|
| `count` | Integer | Количество записей (макс. 500) |
| `offset` | Integer | Смещение для пагинации |
| `type` | String | Тип отказа (HardBounce, SoftBounce и т.д.) |
| `inactive` | Boolean | Только неактивные |
| `emailFilter` | String | Фильтр по email |
| `tag` | String | Фильтр по тегу |
| `messageID` | String | Фильтр по ID сообщения |
| `fromdate` | Date | Начальная дата |
| `todate` | Date | Конечная дата |

## Получение отказа по ID

```java
Bounce bounce = client.getBounce(12345);
System.out.println("Email: " + bounce.getEmail());
System.out.println("Тип: " + bounce.getType());
System.out.println("Описание: " + bounce.getDescription());
```

## Дамп отказа

```java
BounceDump dump = client.getBounceDump(12345);
System.out.println(dump.getBody());
```

## Активация отказа

Снятие блокировки с email-адреса:

```java
Bounce activated = client.activateBounce(12345);
System.out.println("Активирован: " + !activated.getInactive());
```

## Статистика доставки

```java
DeliveryStats stats = client.getDeliveryStats();
System.out.println("Неактивных адресов: " + stats.getInactiveMails());

for (BounceType type : stats.getBounces()) {
    System.out.println(type.getName() + ": " + type.getCount());
}
```
