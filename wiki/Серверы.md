# Серверы

Управление серверами осуществляется через `AccountApiClient`.

## Получение списка серверов

```java
import ru.haskimail.client.data.model.server.Server;
import ru.haskimail.client.data.model.servers.Servers;

AccountApiClient client = Haskimail.getAccountApiClient("аккаунт-токен");

Parameters params = new Parameters();
params.build("count", 50).build("offset", 0);

Servers servers = client.getServers(params);
```

## Получение сервера по ID

```java
Server server = client.getServer(12345);
System.out.println("Имя: " + server.getName());
```

## Создание сервера

```java
Server server = new Server();
server.setName("Новый сервер");
server.setColor("green");
server.setTrackOpens(true);

Server created = client.createServer(server);
```

## Обновление сервера

```java
server.setName("Обновлённый сервер");
client.setServer(server.getId(), server);
```

## Удаление сервера

```java
client.deleteServer(12345);
```
