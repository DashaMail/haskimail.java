# Начало работы

## Установка

### Maven

Добавьте зависимость в `pom.xml`:

```xml
<dependency>
    <groupId>ru.haskimail</groupId>
    <artifactId>haskimail</artifactId>
    <version>1.0.0</version>
</dependency>
```

### Gradle

```groovy
implementation 'ru.haskimail:haskimail:1.0.0'
```

## Получение API-токена

1. Зарегистрируйтесь в [Haskimail](https://haskimail.ru)
2. Создайте сервер в панели управления
3. Скопируйте серверный API-токен

## Инициализация клиента

Библиотека предоставляет два типа клиентов:

### Серверный клиент (ApiClient)

Используется для операций на уровне сервера: отправка писем, работа с шаблонами, статистика и т.д.

```java
import ru.haskimail.Haskimail;
import ru.haskimail.client.ApiClient;

ApiClient client = Haskimail.getApiClient("ваш-серверный-токен");
```

### Клиент аккаунта (AccountApiClient)

Используется для управления серверами, доменами и подписями отправителей.

```java
import ru.haskimail.Haskimail;
import ru.haskimail.client.AccountApiClient;

AccountApiClient client = Haskimail.getAccountApiClient("ваш-аккаунт-токен");
```

## Настройка

### Пользовательский URL

```java
ApiClient client = Haskimail.getApiClient("токен", true, "custom.api.haskimail.ru");
```

### HTTP без SSL

```java
ApiClient client = Haskimail.getApiClient("токен", false);
```

### Таймауты

```java
ApiClient client = Haskimail.getApiClient("токен");
client.setConnectTimeoutSeconds(30);
client.setReadTimeoutSeconds(30);
```
