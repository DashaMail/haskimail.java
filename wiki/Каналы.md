# Каналы (Message Streams)

Каналы позволяют разделять потоки сообщений — транзакционные, маркетинговые и т.д. Каждый канал имеет собственные настройки доставки и отслеживания.

## Получение списка каналов

```java
import ru.haskimail.client.data.model.streams.*;

MessageStreams streams = client.getMessageStreams();
System.out.println("Всего каналов: " + streams.getTotalCount());

for (MessageStream stream : streams.getMessageStreams()) {
    System.out.println(stream.getId() + " - " + stream.getName());
}
```

## Получение канала по ID

```java
MessageStream stream = client.getMessageStream("outbound");
System.out.println("Название: " + stream.getName());
System.out.println("Тип: " + stream.getMessageStreamType());
```

## Создание канала

```java
MessageStream stream = new MessageStream();
stream.setId("marketing");
stream.setName("Маркетинговый канал");
stream.setMessageStreamType("Broadcasts");

MessageStream created = client.createMessageStream(stream);
```

## Обновление канала

```java
stream.setName("Обновлённое название");
stream.setDescription("Канал для маркетинговых рассылок");

client.setMessageStream("marketing", stream);
```

## Архивация канала

```java
MessageStreamArchiveResponse response = client.archiveMessageStream("marketing");
System.out.println("Дата удаления данных: " + response.getExpectedPurgeDate());
```

## Разархивация канала

```java
MessageStreamUnarchiveResponse response = client.unarchiveMessageStream("marketing");
```

## Отправка через канал

```java
Message message = new Message("от@домен.ru", "кому@домен.ru", "Тема", "<p>Тело</p>");
message.setMessageStream("marketing");
client.deliverMessage(message);
```
