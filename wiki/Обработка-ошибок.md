# Обработка ошибок

## Иерархия исключений

```
HaskimailException (базовое)
└── HaskimailHttpException (HTTP-ошибки)
    ├── InvalidAPIKeyException (401)
    ├── InvalidMessageException (422)
    ├── TimeoutException (408)
    ├── InternalServerException (500)
    └── UnknownException (прочие)
```

## Использование

```java
import ru.haskimail.client.exception.*;

try {
    client.deliverMessage(message);
} catch (InvalidAPIKeyException e) {
    System.err.println("Неверный API-ключ: " + e.getMessage());
} catch (InvalidMessageException e) {
    System.err.println("Ошибка валидации: " + e.getMessage());
    System.err.println("Код ошибки API: " + e.getErrorCode());
    System.err.println("HTTP-код: " + e.getStatusCode());
} catch (TimeoutException e) {
    System.err.println("Таймаут запроса");
} catch (InternalServerException e) {
    System.err.println("Внутренняя ошибка сервера");
} catch (HaskimailException e) {
    System.err.println("Ошибка API: " + e.getMessage());
} catch (IOException e) {
    System.err.println("Ошибка сети: " + e.getMessage());
}
```

## Коды ошибок API

| Код ошибки | Описание |
|---|---|
| 0 | Успех |
| 10 | Неверный API-токен |
| 300 | Невалидный email-адрес отправителя |
| 400 | Отправитель не найден |
| 405 | Неверный JSON |
| 406 | Получатель не подтверждён |
| 409 | Ошибка в данных JSON |
